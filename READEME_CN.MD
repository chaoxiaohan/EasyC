## 🚀 快速入门

### 安装指南
1. 克隆仓库
```bash
git clone https://github.com/ophiraShen/EasyC.git
```

2. 安装依赖
```bash
pip install -r requirements.txt
```

3. 启动应用
```bash
python src/main.py
```
